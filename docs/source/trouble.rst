Локализация неисправностей
==========================

Настоящий раздел предназначен для помощи службе технической поддержки при локализации и первичном анализе неисправностей, возникающих в процессе эксплуатации услуги **V2TextMail**.

Общие рекомендации по диагностике
---------------------------------

При возникновении неисправностей в работе услуги **V2textMail** рекомендуется выполнять диагностику по следующему порядку:

1. Проверить статус подключения услуги для абонента.
2. Убедиться в наличии голосового сообщения и корректности его записи.
3. Проверить наличие ошибок передачи данных в почтовую систему оператора.
4. При необходимости проанализировать журналы компонентов системы **VoiceBox**.
5. В случае подтверждения корректной передачи данных проинформировать абонента о том, что дальнейшая доставка сообщения осуществляется почтовой системой оператора.

Границы ответственности
-----------------------

Система **VoiceBox** и услуга **V2TextMail** отвечают за обработку вызова, запись и транскрибацию голосового сообщения, а также передачу данных в почтовую систему оператора.

После подтверждения успешной передачи данных дальнейшая обработка и доставка электронного письма находятся в зоне ответственности оператора и 
не относится к зоне ответственности службы технической поддержки системы **VoiceBox**.
