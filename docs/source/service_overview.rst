Общая информация об услуге
===========================

Наименование услуги
--------------------
**V2TextMail** (Voice-to-Text Mail) – услуга автоматической транскрибации голосовых сообщений, оставленных абоненту при переадресации вызова на сервис системы **VoiceBox**, с 
последующей отправкой текстовой версии сообщения на электронную почту абонента.

Услуга обеспечивает преобразование речевого сообщения вызывающего абонента в текстовый формат и передачу следующих данных по электронной почте:

* Номер вызывающего абонента;
* Время вызова;
* Текст транскрибированного голосового сообщения.

Цель услуги
------------
Цель услуги для клиента – обеспечение альтернативного способа получения информации, содержащейся в голосовых сообщениях, оставленных при переадресации вызовов, в текстовом формате
в условиях ограниченной возможности прослушивания аудио.

Цель услуги для оператора и поставщика сервиса – расширение функциональных возможностей системы **VoiceBox**, 
за счет предоставления дополнительного способа доставки информации абонентам, без существенных изменений существующей инфраструктуры.

Связь с системой VoiceBox
--------------------------
Услуга **V2TextMail** является функциональным расширением системы **VoiceBox** и использует ее инфраструктурные и программные компоненты для обработки 
вызовов, записи голосовых сообщений и взаимодействия с сетевыми элементами оператора.
