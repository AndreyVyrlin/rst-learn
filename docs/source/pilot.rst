Особенности пилотного проекта
=============================

Общая информация о пилотном проекте
-----------------------------------
Пилотный проект услуги **V2TextMail** реализован на сети оператора ООО «Ромашка Телеком» в 
столичном регионе и предназначен для оценки работоспособности сервиса и анализа пользовательского опыта в условиях реальной эксплуатации.

В рамках пилотного проекта услуга предоставляется ограниченной группе абонентов оператора на безвозмездной основе. Подключение услуги для участников пилотного проекта выполнено 
централизовано без необходимости дополнительных действий со стороны абонентов.

Продолжительность пилотного проекта составляет 90 календарных дней. По завершении пилотного периода услуга автоматически отключается для всех участников тестовой группы, 
после чего абонентам направляется информация о возможности повторного подключения услуги на коммерческих условиях.

Ограничения пилотного проекта
-----------------------------
В рамках пилотного проекта установлены следующие ограничения и особенности эксплуатации услуги **V2TextMail**:

* Услуга доступна исключительно абонентам, включенным в тестовую группу, численность которой ограничена 5000 абонентов.
* Пилотный проект реализуется только в столичном регионе и не распространяется на другие регионы сети оператора.
* Отключение услуги абонентам в период пилотной эксплуатации возможно только через обращение в службу технической поддержки компании SIfox.
* Для целей увеличения объема тестовых данных в период пилота используется измененное значение тайм-аута переадресации при отсутствии ответа вызываемого абонента.
* В рамках пилотного проекта не предусмотрено географическое резервирование компонентов услуги между регионами.
* Параметры пилотной эксплуатации носят временный характер и подлежат возврату к штатным значениям по завершении пилотного проекта.
