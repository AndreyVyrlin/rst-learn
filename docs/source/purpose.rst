Назначение документа
====================

Настоящий документ предназначен для использования сотрудниками службы технической поддержки Sifox при эксплуатации услуги **V2TextMail**
на сетях операторов связи.

Отдельные разделы документа могут быть использованы при подготовке внешней документации для заказчиков и отраслевого регулятора.
